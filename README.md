# Задание 1

### Текстовые файлы

В файле найти сумму квадратов элементов.

### Алгоритм решения

1. Реализован метод для генерации текстового файла со случайными целыми числами, записанными по одному в строке. Параметры метода: имя файла, количество чисел, минимальное и максимальное значения.
2. Реализован метод, который:
   - Открывает указанный текстовый файл для чтения
   - Построчно считывает содержимое файла
   - Преобразует каждую строку в целочисленное значение
   - Вычисляет квадрат каждого числа и добавляет к накапливаемой сумме
   - Возвращает итоговую сумму квадратов

# Задание 2 

### Текстовые файлы

Вычислить произведение элементов.

### Алгоритм решения

1. Реализован метод для генерации текстового файла со случайными целыми числами, записанными по несколько чисел в строке через пробел. Параметры метода: путь к файлу, количество строк, количество чисел в строке, минимальное и максимальное значения.
2. Реализован метод, который:
   - Открывает указанный текстовый файл для чтения
   - Построчно считывает содержимое файла
   - Разделяет каждую строку на подстроки с использованием пробелов и символов табуляции в качестве разделителей
   - Преобразует каждую подстроку в целочисленное значение
   - Вычисляет произведение всех чисел
   - При отсутствии чисел в файле возвращает 0

# Задание 3

### Текстовые файлы

Переписать в другой файл строки, имеющие заданную длину m.

### Алгоритм решения

1. Реализован метод, который:
   - Принимает путь к исходному файлу, путь к файлу назначения и целевое значение длины строки m
   - Открывает исходный файл для чтения
   - Открывает файл назначения для записи 
   - Построчно считывает исходный файл
   - Для каждой строки сравнивает её длину с заданным значением m
   - При совпадении длины записывает строку в файл назначения

# Задание 4

### Бинарные файлы

Вычислить произведение нечетных отрицательных компонент файла

### Алгоритм решения

1. Реализован метод для генерации бинарного файла со случайными целыми числами. Параметры метода: путь к файлу, количество чисел, минимальное и максимальное значения
2. Реализован метод, который:
   - Открывает бинарный файл для чтения
   - Последовательно считывает все целочисленные значения типа int до достижения конца файла
   - Для каждого считанного числа проверяет два условия: число отрицательное и число нечётное (определяется как Math.Abs(number) % 2 == 1)
   - При выполнении обоих условий умножает число на накапливаемое произведение
   - При отсутствии подходящих чисел возвращает 0
